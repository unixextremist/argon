# what is it?

- argon is a *meta?* package manager for git repositories

![showcase](showcase.gif)

# how it works?

argon works by 

- cloning the repository
- auto detecting the build system
- building
- moving to $PATH

# dependencies 

- golang
- pkgconfig
- git
- patch
- less
- build systems like make cmake cargo etc

# building 

- just run make
